ABOUT

CMDR Strict's Void Opal Tool began as a summer project to learn the Django web framework and to brush up on my Python. While perhaps not the ideal framework for this kind of application, the process of building the app was challenging, rewarding, and illuminating.

The raw data used to construct the PostgreSQL database used by this program is sourced from EDSM.net & EDDB.io.

Turns out 105GB files of data concerning an entire galaxy is a lot to parse through and wrangle!

Queries for the latest information are made to EDSM using their API.

Additional market information not directly exposed by the EDSM API is obtained by webscraping individual station listings on EDDB.io
